spring-boot-starter-rockermq。

支持jdk版本为1.8

### 如何发布dubbo服务

* 添加依赖:

```xml
    <dependency>
		<groupId>org.alittlebitch.rocketmq</groupId>
		<artifactId>spring-boot-starter-rocketmq</artifactId>
    	<version>1.0.0</version>
    </dependency>
```

Demo可参考test包下，稍后会继续补充使用文档。
